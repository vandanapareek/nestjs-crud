## NestJS CRUD

### Categories
Category includes following fields:
name, slug, description

### Products
One category has multiple products:
name, slug, description, price, quantity, category.


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
